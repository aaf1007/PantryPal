# PantryPal

Monorepo for the PantryPal application, containing a React client and a Spring Boot server.

## Structure

- `client/` – Frontend React app (see `client/README.md` for details and scripts).
- `server/` – Backend code.
  - `server/pantry-pal/` – Spring Boot application.

## Development

- Run frontend commands from `client/`.
- Run backend commands (e.g. `mvn spring-boot:run`) from `server/pantry-pal/`.
